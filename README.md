# Drupal D&D Campaign Manager

**Un’app in Drupal per Dungeons & Dragons (D&D) che permette a GM e giocatori di gestire campagne, schede personaggio e aggiornamenti della campagna con permessi personalizzati.**

## Funzionalità principali

- Creazione e gestione di campagne da parte del GM.  
- Creazione e modifica delle proprie schede personaggio per i giocatori.  
- Aggiornamenti della campagna visibili a tutti i partecipanti.  
- Visualizzazione filtrata tramite Views.  
- Possibile integrazione con dati ufficiali D&D tramite API (Open5e).  

## Ruoli

- **GM:** controlla la campagna, tutte le schede e gli aggiornamenti.  
- **Giocatori:** gestiscono le proprie schede e visualizzano contenuti della campagna.  
- **Visitatori (opzionale):** visualizzano contenuti pubblici.  

## Setup rapido

1. Installa Drupal.  
2. Abilita moduli: Entity Reference, Views, Permissions/Rules.  
3. Crea ruoli GM e Giocatore.  
4. Crea tipi di contenuto: Campagna, Scheda Personaggio, Aggiornamento.  
5. Configura Views per la visualizzazione dei contenuti.  
6. (Opzionale) Integra API D&D per popolamento automatico di classi, razze e mostri.
